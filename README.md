## Setup Environment

With anaconda
```bash
  conda create -n name-ds python=3.9
  conda activate name-ds
```
With venv
```bash
  py -m venv namevenv
  namevenv\Scripts\activate
```
after your activate you can install package with
```bash
  pip install -r requerments.txt
```
